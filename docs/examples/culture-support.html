Culture Support

The usage of a specific culture is also possible and can be done by using parameter provider.

For the following example consider the usage of German culture.

Copy
Decimal price = 123456.789M;
DateTime date = DateTime.Parse("2020-10-29T23:05:17.123");
CultureInfo culture = CultureInfo.CreateSpecificCulture("de-DE");
Template.Format(culture, "Price: {price:C2} Date: {date:dddd, dd. MMMM yyyy}", price, date);

The result would look like "Price: 123.456,79 € Date: Donnerstag, 29. Oktober 2020".

In contrast to the above, consider the usage of US culture.

Copy
Decimal price = 123456.789M;
DateTime date = DateTime.Parse("2020-10-29T23:05:17.123");
CultureInfo culture = CultureInfo.CreateSpecificCulture("en-US");
Template.Format(culture, "Price: {price:C2} Date: {date:dddd, dd. MMMM yyyy}", price, date);

The result would look like "Price: $123,456.79 Date: Thursday, 29. October 2020".

Please note, the format string is unchanged in both cases but the output differs.
