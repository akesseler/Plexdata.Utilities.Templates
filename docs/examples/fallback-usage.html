Fallback Usage

As already mentioned, missing arguments are usually replaced by their “Marker”, which is the complete formatting instruction. Following example shows what is meant in detail.

Copy
Options options = new Options();

Template.Format(options, "formatted string with \"{value,10:N3}\" data");

The result would look like "formatted string with "{value,10:N3}" data".

But sometimes it might be useful to have some kind of default string replacement, for example something like “unused”. Therefore, it is possible to customize the formatter by providing a value for the Fallback. See example below how to achieve something like that.

Copy
Options options = new Options() { Fallback = "unused" };

Template.Format(options, "formatted string with \"{value,10:N3}\" data");

With the same format instruction, now the result would look like "formatted string with "unused" data".
