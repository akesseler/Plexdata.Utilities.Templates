Format Escaping

Sometimes it might be useful to have curly braces within the formatted result. For this purpose the usage of two curly braces is recommended, which causes the formatter to take the content as it is.

Here an example for an index–based formatting with system types but with escaped content.

Copy
Template.Format("text {1} text {{escaped}} text {2}", 111, "ABC", 333.33);

The result would look like "text ABC text {escaped} text 333.33".

The same example as above but for label–based formatting.

Copy
Template.Format("text {ValA} text {{escaped}} text {ValC}", 111, "ABC", 333.33);

The result would look like "text 111 text {escaped} text ABC".
