Length Truncation

The limitation of the length of result strings is only possible by using an instance of class Options.

The following example shows how to limit the result length to let’s say 30 characters.

Copy
Options options = new Options() { Maximum = 30 };

Template.Format(options, "formatted string with a long length");

The result would look like "formatted string with a lon...". It has been truncated and three dots were appended. Note that length truncation takes place for the whole result. This includes the three dots as well.

Against this background, this behaviour implies some side effects that come into view near the lower limit. See next example for what is meant in detail.

Copy
Options options = new Options();

options.Maximum = 1;
result = Template.Format(options, "formatted string with a long length"); // => "f"

options.Maximum = 2;
result = Template.Format(options, "formatted string with a long length"); // => "fo"

options.Maximum = 3;
result = Template.Format(options, "formatted string with a long length"); // => "for"

options.Maximum = 4;
result = Template.Format(options, "formatted string with a long length"); // => "f..."

options.Maximum = 5;
result = Template.Format(options, "formatted string with a long length"); // => "fo..."
