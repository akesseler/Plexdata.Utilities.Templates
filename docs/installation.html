Installation

The binary files of the Plexdata Template Formatter has been published on NuGet.org and can be obtained from there. On that page search for plexdata and you will find all packages, including this package.

How to manually install this package is explained on NuGet.org. But for sure it doesn’t make any sense to just download the libraries. It’s indeed a better idea to include it in your projects.

Below find the steps necessary to use the template formatter together with Visual Studio.

  • Create or open a .NET Framework project, or a .NET Standard project, or a .NET Core project.
  • Open menu Tools → NuGet Package Manager and choose command Manage NuGet Packages for Solution…
    • Alternatively, right‑click the project in the Solution Explorer and choose command Manage NuGet Packages…
  • Switch to Browse page and in the search box just type plexdata.
  • From the shown list select for example package Plexdata.Utilities.Templates. Perhaps you may have to tick the check–box for all projects you want to assign the chosen package.
  • Finally, click button [Install] at the right and confirm the dialog box. That’s it. All dependent libraries are installed automatically.

All sources as well as all binary files are available on GitHub. Thus, please feel free to browse the template formatter project on GitHub.
