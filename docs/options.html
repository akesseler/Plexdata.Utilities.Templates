Options

The Plexdata Template Formatter provides some settings to allow a minimum setup of template formatting. This section wants to give a short overview about these options and their meaning.

Class Options in namespace Plexdata.Utilities.Formatting can be used to configure the formatting behaviour. An instance of this class is then used together with a call of one of the overloads of Template.Format(). How to do this is shown in this very short and abstract code snippet.

Copy
Options options = new Options()
{
    // ...
};

Template.Format(options, ...);

Provider

Property Provider is used to perform value formatting according to additional format instructions. It can be changed by users into any class derived from IFormatProvider. No provider–specific formatting is performed if this property is set to null. Its default value is set to CultureInfo.InvariantCulture.

Serializer

Property Serializer allows users to apply a custom–specific serializer for complex data types. Its default value is set to null.

Such a custom class must be derived from interface IArgumentSerializer in namespace Plexdata.Utilities.Formatting.Interfaces. As well, such a class must be able to serialize every custom–specific data type. Furthermore, this serializer is not used for build–in system types. An internal fallback serializer is used if no other argument serializer has been applied.

Maximum

Property Maximum allows users to apply a limitation of the maximum result length. Its default value is set to Int32.MaxValue. Please keep in mind, any value less than one or greater than Int32.MaxValue-1 is treated as “unlimited”. More information about using this property can be found in the examples.

IsLimited

Property IsLimited is nothing else but a convenient getter to determine whether result length limitation is applied or not. It returns true if the Maximum is set to a range of 1 up to Int32.MaxValue-1. Otherwise false is returned.

Fallback

Property Fallback allows to define a custom–specific name to be used as alternative label for unresolved arguments. Its default value is set to null. This causes the formatter to show the whole formatting string for each argument that could not be resolved. Otherwise the value of this property is shown. More information about using this property can be found in the examples.

IsFallback

Property IsFallback is nothing else but a convenient getter to determine whether the value of property Fallback should be used or not. It returns false if the value of property Fallback is null. Otherwise true is returned.

Default

Class Options provides a public field named Default to obtain the standard settings. The value of this field is used if no other settings are provided and uses all of the default values as shown above.
